# Knowra (知识探索与智能助理)

Knowra 是一款基于大语言模型（LLM）与 RAG (检索增强生成) 技术的现代桌面知识库管理与沉浸式对话工具。它结合了本地离线数据存储与在线云端 AI 能力，协助用户通过自然对话迅速从海量学习与研究材料中定位信息。

## 🪄 核心界面与理念
通过参考各大前沿知识管理方案，Knowra 目前采用 **四级联动式多窗体布局** 与 **Claude 风格温润白** 主题设计：
- **第一级极简导航**: 全局状态切换点与配置中心。
- **第二级分区清单**: 知识空间 (Spaces) 侧滑菜单，将不同课题内容物理隔离，保持专注，可随用随合。
- **第三级内容总览**: 当前知识分片内的所有长文、笔记与附件详情支持拖拽导入极速建排。
- **第四级大模型大脑**: 基于 RAG 技术直接结合目前第三级窗口内的上下文为您作流式解答，并能溯源展示原始资料卡片。

## 🚀 技术底座
- **展现层**: 基于 PyQt6 手感细腻且具备严谨 QThread 与信号槽机制分离阻断假死的主窗体。
- **召回层**: SQLite FTS5 (结巴分词倒排) 加上 ChromaDB 向量混合检索，依据 RRF 计算打分进行混合搜索排序。
- **逻辑层**: 具备安全系统密钥环 (KeyRing) 保管的大模型连接层，支持文档无边界分块切割 (Token-Aware Chunker)。

## 🛠️ 开始使用
1. 请先确保本地依赖完整 (包含 PyQt6, ChromaDB, PyMuPDF 等)：
   ```bash
   pip install -r requirements.txt
   ```
2. 直接运行位于目录树的启动文件:
   ```bash
   python src/main.py
   ```
3. 点击主界面左下角的 ⚙️ 齿轮图标预先配置由平台提供的大模型 `API_KEY`，点击上方「新建空间」即可任意拖拽您的文件与大模型无缝交流。
